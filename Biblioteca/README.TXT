Sistema de Biblioteca

## Descrição
Este projeto é um sistema de gerenciamento de uma biblioteca, onde é possível adicionar livros, ver listagem de livros etc.

## Tecnologias Utilizadas
- Java
- PostgreSQL
- JDBC

## Instruções de Instalação
1. Clone o repositório.
2. Importe o projeto em sua IDE favorita.
3. Execute o script `database.sql` no PostgreSQL para criar o banco de dados.
4. Configure as informações de conexão com o banco de dados no arquivo `DatabaseConnection.java`.

## Instruções de Uso
- Compile e execute a classe `Main.java` para iniciar a aplicação.
- Siga as instruções no console para utilizar as funcionalidades disponíveis.

## Estrutura de Diretórios
- /src: Contém o código-fonte Java.
- /sql: Scripts SQL para criação do banco de dados.
